# harshathkasim.com

Personal website for Harshath Kasim. Currently a placeholder landing page; a full portfolio is in progress.

## Stack
- Static `index.html` (no build step)
- Deployed on Vercel, auto-deploys on push to `main`
- Domain: harshathkasim.com (GoDaddy)

## Local preview
Open `index.html` in a browser, or run a static server:

```bash
python -m http.server 8000
```
